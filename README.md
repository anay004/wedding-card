# Wedding Invitation Website

This is a romantic invitation-style website inspired by the reference screens:

- tap-to-open envelope/kham reveal
- animated wax seal
- wedding invitation message
- falling petals
- music toggle
- auto photo slideshow
- main invitation card image section
- venue and map-style section
- ceremony card image section
- maroon closing / RSVP section

## Replace These Assets

Create an `assets` folder next to `index.html`, then add:

- `music.mp3`
- `main-invitation-card.jpg`
- `ceremony-1.jpg`
- `venue.jpg`
- `photo-1.jpg`
- `photo-2.jpg`
- `photo-3.jpg`
- `photo-4.jpg`
- `photo-5.jpg`
- `photo-6.jpg`
- `photo-7.jpg`
- `photo-8.jpg`

## Edit Names And Details

Open `index.html` and replace:

- `Bride Name`
- `Groom Name`
- parent names
- date
- venue name and address
- RSVP names
- Google Maps link in the `Get Directions` button

## Open

Open `index.html` in a browser, or run a simple local server from this folder.
